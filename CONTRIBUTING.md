# Contributing to "Invest in Yourself" 🚀

First off, thank you for considering contributing to this roadmap! The goal is to build the ultimate, open-source guide for personal, professional, and financial growth.

## How can you contribute?

### 1. Suggesting a New Resource (Link/Book/Course)
If you know an amazing free resource that changed your life:
- Check if it's already in the `README.md` or `README_EN.md`.
- Open an **Issue** using the "Suggest Resource" template, OR
- Submit a **Pull Request** directly.

### 2. Fixing Broken Links or Typos
If a link is dead or you found a typo:
- Open an **Issue** to report it, OR
- Submit a **Pull Request** to fix it.

## Pull Request Process
1. **Fork** the repository.
2. Create a new branch: `git checkout -b feature/new-resource`
3. Add your resource. **Important:** If you add something to the Greek `README.md`, please add it to the English `README_EN.md` as well (and vice versa).
4. Keep the descriptions short, actionable, and aligned with the "Core Philosophy".
5. Commit your changes: `git commit -m "Add [Resource Name] to [Category]"`
6. Push to the branch: `git push origin feature/new-resource`
7. Submit a **Pull Request**.

Let's build the ultimate roadmap together! 🤝
